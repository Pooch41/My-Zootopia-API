# My Zootopia(API Version)

Fetches ANIMAL API provided, converts information to HTML body - animals template required.
Catches and reports web errors when fetching APIs.

## Installation

Check requirements.txt for required libraries. Install as needed with pip3 install. Confirm that
all libraries are present before starting usage.

## Usage

To use this project, run the following command - `python animals_web_generator.py`,
with HTML template available. Generates separate HTML document with desired results.
